# Changelog

All notable changes to DriveDock will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added
- Initial release of DriveDock
- Native macOS app built with Swift and SwiftUI
- Google OAuth 2.0 authentication with multi-account support
- Drag-and-drop file and folder uploads
- Google Drive API v3 integration
- Parallel upload engine with adaptive concurrency
- Resumable upload support for large files
- Upload queue management (pause, resume, cancel, retry)
- Destination picker with My Drive and Shared Drive browsing
- Folder structure preservation during uploads
- Upload history with detailed completion reports
- Menu bar helper with quick status and controls
- Background upload support
- Native macOS notifications
- Duplicate file handling (keep both, rename, skip)
- Secure token storage in macOS Keychain
- Settings panel with General, Upload, Account, and Privacy options
- Smart error recovery with exponential backoff
- Dark mode and Light mode support
